Collection Data Types

  • A collection, or a list, is a data structure that can store multiple values. Two popular collections are arrays and dictionaries.

  • Arrays

    • Arrays store an ordered list of values.

    • Array values are ordered by index number with the 0-th index referring to the first item in the array.

    • Array values are retrieved by using bracket notation [ ] after the array, and providing an index number inside the brackets.

    • Use arrays when you have a list of values that are of similar data type and you want to run the same code on each of those values.

      [1, 2, 3, 4]; // creates an array literal

      var numbers = [1, 2, 3, 4]; // stores an array to a variable

      numbers[0]; // returns the first item in the array, which is 1

      numbers[1]; // returns the second item in the array, which is 2

      numbers.length; // .length is a built-in property of an array that returns the number of items in the array. In this case it's 4

    • There are several methods built in to an array that can help you interface with the array. Some of them are:

      • .push(element) - appends an element to the end of the array

      • .pop() - removes the last element from an array

        var numbers = [1, 2, 3, 4];

        numbers.push("hello"); // the array is now [1, 2, 3, 4, "hello"]

        numbers.pop(); // the array is now [1, 2, 3, 4]

        // You can also set a value in an array by accessing the index where you want to store the value at:

        numbers[numbers.length] = "world"; // add "world" to end of array

    • For a comprehensive list of array methods and properties, check out the following URL: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array

      You should review .forEach(), .join(), .slice(), .splice()

  • Dictionary

    • Dictionary is an unordered collection of values where each value is mapped to a unique key.

    • To set or get a value, you use bracket notation, [ ], or dot notation, ., after the dictionary.

    • In JavaScript, a dictionary is represented by an object literal, and a key from a dictionary is commonly referred to as a property of the object:

      // This dictionary (object) stored in a variable called person.
      // Each key-value pair syntax is written as key: value
      // The key-value pairs are separated by commas. var person = { name: "Chris", // the property is name and the value is "Chris" age: 55 };

      person.name; // dot notation: this evaluates to "Chris"

      person["name"]; // square bracket notation: this evaluates to "Chris"

      var key = "name";

      person[key]; // this evaluates to "Chris"

    • Even after you have created the JavaScript object, you can still add new key value pairs:

      person.skills = ["HTML", "CSS", "JavaScript"]; // Adding a new property called skills that stores an array of Strings.

    • You can remove an object's property using the delete keyword:

      delete person.age; // delete the age property from person object

      person.age; // person.age now evaluates to undefined
