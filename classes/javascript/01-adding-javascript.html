Adding JavaScript to your web pages

  1. Embed JavaScript code in a web page

    • Create a script element and add JavaScript code inside its opening and closing tags.

      Place this inside an HTML file:

      <script> alert("Hi there!"); </script>

  2. Reference an external JavaScript file from within the web page.

    • Create a script element and add a src attribute that contains the URL of a JavaScript file.

      Place this inside an HTML file:

      <script src="js/jquery.js"></script>

    • All JavaScript filenames should end with a .js suffix.

  3. Comments

    • You can add comments to your JavaScript code. Comments serve as notes to programmers to remind them the intent behind a particular segment of code. Machines do not understand comments and will not try to interpret them.

    • You can add a single-line comment or a mult-line comment.

    • To create a single line comment you begin it with a //

      Single-line comment can begin on its own line or be the last item on a line of code.

      // A single line comment on its own line

      alert("hello world"); // A single line comment following a JavaScript statement

    • To create a multi-line comment you wrapped the comment in between /* and */

      You can create as many lines as you need for your comment when using multi-line comments.

      /* A multi-line comment using only one line */

      /* a comment here

      a comment here

      a comment here */

      alert("hello world"); // each code statement should end with a semicolon
