jQuery

  • jQuery (http://jquery.com/) is a JavaScript library file created by John Resig (http://ejohn.org/) back in 2006.

  • Though it has many useful features, jQuery's main feature is handling the the differences in the DOM API across the major browsers, so that you don't need to worry about these discrepancies.

  • Many websites have incorporated the jQuery library file in their pages. You add jQuery to your web pages the same way you would add any external JavaScript file:

    • Download the file to the web server hosting your site. You can download jQuery from the jQuery download page: http://jquery.com/download/

    • Use a CDN (Content Delivery Network http://en.wikipedia.org/wiki/Content_delivery_network). A list of CDNs hosting the jQuery file is provided on the jQuery download page.

    • Either way, you will still need to add a script element to your web page with the src attribute pointing to the location of the jQuery file:

    Local to the web server:

    <script src="js/jquery-1.11.2.min.js"></script>

    or a URL to a CDN:

    <script src="//code.jquery.com/jquery-1.11.2.min.js"></script>

  • You start using jQuery to work with the DOM by either:

    1. Create an element to add to the DOM tree, or

    2. Find an element on the DOM tree.

  • For the complete jQuery API reference, go to the following URL: http://api.jquery.com/

  • The jQuery() function

    • The jQuery() function can also be written as $()

    • Invoking the jQuery function will return a jQuery object. You will use the methods from this object to interact with your web pages.

    • The jQuery function can take a string as an argument. This string can either be a CSS selector-type string that will query the DOM tree for elements that match the selector, or the string can contain HTML that jQuery will create. Either way, the returned jQuery object will contain those element which you can then interact with using jQuery methods and properties.

    • We will refer to the element(s) a jQuery object contains as a matched set or a jQuery selection.

    1. Use the jQuery function to find elements on a web page:

      // Pass a CSS selector-type string to the function.

      var $divs = $('div'); // Search for all <div> elements on the web page. The returned jQuery object will contain all the <div>s found.

      var $spans = $('div.box p span');

      // You can now call methods on the jQuery object that will in turn work on the found element(s):

      $divs.addClass('box').css({"background-color": "red", "padding": "50px"});

    2. Use the jQuery function to create HTML element(s):

      // This creates a <div> element and the returned jQuery object is stored in a variable.

      var $newlyCreatedDiv = $('<div>');

      var $someNewlyCreatedHTML = $('<div><p>hello world!</p></div>');

      // The new elements will need to be added to the DOM tree:

      $newlyCreatedDiv.appendTo("article"); // Insert the newly created element(s) to the end of each <article> element.

      $('article').append($someNewlyCreatedHTML); // Insert thew newly created element(s) to the end of each <article> element.

    3. Use jQuery function to invoke a function (a callback) when the DOM is ready:

      // Pass a callback function as an argument to the jQuery function. $(function () { // This callback function is invoked when DOM is ready. });

  • jQuery methods

    Most jQuery methods returns a jQuery object, which makes method chaining possible:

    // This is a single statement broken into three lines for readability:

    $('div') // This evaluates to a jQuery object .children() // This evaluates to a jQuery object .addClass('fade'); // This evaluates to a jQuery object

    The list of jQuery methods is extensive, and covers different areas of the DOM. Here are a few you should be familiar with if you will be using jQuery in your web pages:

    Traversing the DOM:

    For complete list of Traversing the DOM jQuery methods, go to the following URL: http://api.jquery.com/category/traversing/

    • .children([selector]): Get the children of each element in the matched set. You can pass an optional selector to filter the child elements.

    • .next([selector]): Get the immediately following sibling of each element in the matched set. You can pass an optional selector that each sibling has to match.

    • .prev([selector]): Get the immediately preceding sibling of each element in the matched set. You can pass an optional selector that each sibling has to match.

    Manipulating the DOM:

    For complete list of Manipulating the DOM jQuery methods, go to the following URL: http://api.jquery.com/category/manipulation/

    • .addClass(className): Add one or more space-separated classes to be added to the class attribute for each element in the matched set.

    • .append(content[, content]): Insert content to the end of each element in the matched set.

    • .appendTo(target): Insert every element in the matched set to the end of the target. Target can be a selector, element, HTML string, array of elements, or jQuery object.

    • .html(): Get the HTML contents (in string format) of the first element in the matched set.

    • .html(htmlString): Set the HTML contents of each elememt in the matched set with the htmlString argument.

    • .removeClass([className(s)]): Removes classNames(s) from each element in matched set. If no argument is provided, removes all classNames from matched set.

    • .text(): Get the combined text content of each element (and its descendants) in the matched set.

    • .text(text): Set the content of each element in the matched set to the specified text.
