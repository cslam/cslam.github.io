JavaScript Built-in Objects

  • JavaScript provides built-in objects that are in the global scope, meaning you can access them anywhere in your code.

  • You should be aware of setTimeout(), setInterval(), String, Date and Math.

  • For a list of the standard built-in objects, you can go to the following URL: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects

  • Timers

    JavaScript timers allow you to run a piece of code at a later time.

    setTimeout()

    • setTimeout(func, delay): calls a function after a certain delay (in milliseconds).

    • setTimeout returns a timeoutID which you can pass as argument to clearTimeout() to cancel the timer before it runs.

      function showAlert() { alert("hello world!"); }

      var timeoutId = setTimeout(showAlert, 1000); // invoke showAlert after 1000 milliseconds or 1 second.

      clearTimeout(timeoutId); // you can stop the timer if it hasn't started yet.

    setInterval()

    • setInterval(func, delay): calls a function repeatedly, with a delay between each call.

    • setInterval returns a timeoutID which you can pass as argument to clearInterval() to cancel the timer before the next run.

      function printToConsole() { console.log("hello world!"); }

      var timeoutId = setInterval(printToConsole, 1000); // invoke printToConsole after 1000 milliseconds or 1 second.

      clearInterval(timeoutId); // you can stop the timer before the next run.

  • String

    • The String data type provides methods you can use on an instance of a String.

      var str1 = "Today is Wednesday."

      str1.length; // returns the number of characters in the string: 19

      // Each character in a string is indexed like an array would index its elements.

      str1.indexOf("day"); // returns the index for the first occurence of the string "day" - in this case the return value is 2.

      str1.indexOf("day", 3); // .indexOf() takes an optional index parameter. This index parameter tells the method which index to start the search on - in this case the return value will be 15.

      str1.indexOf("foo"); // returns -1 if a match cannot be found.

      str1.toLowerCase(); // returns a string with the letters being lowercase.

      str1.toUpperCase(); // returns a string with the letters being uppercase.

      For more information on Strings check out the following URL: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String

  • Date

    • The Date constructor creates a Date instance. Here are the ways you can call the constructor:

      new Date(); // creates a date instance with the current date and time based on the system settings

      new Date(value); // argument is an integer representing the number of milliseconds since 1 January 1970 00:00:00 UTC

      new Date(dateString); // argument is a string in a format that the constructor recognizes: RFC2822 or ISO 8601

      new Date(year, month[, day[, hour[, minutes[, seconds[, milliseconds]]]]]); // at least 2 arguments representing the year and month (0 to 11), and optional arguments for the rest (the square brackets means the arguments within them are optional, you don't actually type in the square brackets).

    • A Date instance has many methods, you can check them out at this URL: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date

    • To compare two dates, you should call the .getTime() method. It returns the number of milliseconds for that date instance since January 1st 1970 00:00:00 UTC.
