Object Constructors

  • An object constructor provides a template for creating objects of the same type. It uses a function for creating the template.

  • To create an object from a constructor function, you insert the keyword new before the constructor function invocation:

    // The name used for a constructor function should be in camel-cased format with the first letter capitalized.

    function Vehicle(speed) { // Inside a constructor, the keyword "this" refers to the newly created instance created by the "new" keyword.
    // You add properties to the instance by using the "this" object inside the constructor: any instances of Vehicle will now have a currentSpeed property. this.currentSpeed = speed || 0; }

    var myCar = new Vehicle(); // myCar is an instance (object) of type Vehicle

    var anotherCar = new Vehicle(5); // anotherCar is also an instance of type Vehicle

    myCar.currentSpeed; // evaluates to 0

    anotherCar.currentSpeed; // evaluates to 5

  • Instance methods

    • You can add methods (functions) to a constructor and all instances creatd by the constructor will have access to those methods. The most common way to do this is to add the methods to the constructor's .prototype property:

      function Vehicle(speed) { this.currentSpeed = speed || 0; }

      // Adding a method called description to the constructor:

      Vehicle.prototype.description = function () { return "The current speed is: " + this.currentSpeed; }

      var myCar = new Vehicle(); // myCar is an instance (object) of type Vehicle

      var anotherCar = new Vehicle(5); // anotherCar is also an instance of type Vehicle

      myCar.description(); // evaluates to "The current speed is: 0"

      anotherCar.description(); // evaluates to "The current speed is: 5"
