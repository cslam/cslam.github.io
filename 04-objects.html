Objects

  • An object contains values that are referenced by their unique property names.

  • These values can be basic data types (string, number, boolean, etc.), objects, and functions.

  • Objects can be used to model data structures that are more complex than the ones JavaScript provides by default.

    // A data type that represents a Vehicle:
    // We are creating an object using literal notation, later we will learn how to use a constructor instead:
    var myCar = { currentSpeed: 0.0 };

  • To retrieve the value of an object's property, you get a reference to the object (e.g., a variable pointing to the object), and append dot or bracket notation after it:

    // Using dot notation:
    myCar.currentSpeed; // returns the value of currentSpeed

    // Using bracket notation: (the property name has to be in String format):
    myCar["currentSpeed"]; // returns the value of currentSpeed

    // Using dot notation:
    myCar.currentSpeed = 50.0; // sets the value of currentSpeed to 50.0

    // Using bracket notation: (the property name has to be in String format):
    myCar["currentSpeed"] = 50.0; // sets the value of currentSpeed to 50.0

  • Functions that are defined within an object are commonly known as methods of the object.

  • Inside every method, you can use a special keyword called this. The this keyword will be the current object that this method is invoked on.

    var myCar = { currentSpeed: 0.0, move: function () { this.currentSpeed = 20.0 } };

    myCar.move(); // invoke .move() on object myCar, that means the this keyword inside move will be the myCar object
