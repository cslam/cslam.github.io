Functions

  • To create a function:

    1. Use the keyword function,

    2. Followed by an identifier (a name for the function),

    3. Followed by a pair of parentheses containing optional parameters separated by commas,

    4. and finally, the function's body wrapped in a set of opened and closed curly braces: { }

  • You should create a function if you want to save a group of statements (code) that performs a specific task. That way you can re-use that code by calling the function over and over again.

  • You define parameters for your function if it needs additional information from the caller of the function in order for the function to do its work.

  • You invoke a function by typing its name followed by a pair of parentheses. If the function defines parameter(s), you pass argument(s) inside those parentheses and make sure they match the order of the defined parameter(s).

  • Example:

    /*
    The name for this function is "area".
    It takes two parameters named "width" and "height" (Give your paremeters descriptive names).
    These parameter names act like variables and can be used only inside the function.
    */     function area(width, height) { // You can optionally return a value to the caller of the function using the "return" keyword. return width * height; }

    /*
    - To call (invoke) a function, you include the name of the function followed by a set of parentheses.
    - You should pass as many arguments as there are parameters defined for that function.
    - If the function returns a value, you usually would want to save that value to a variable.
    */

    var areaOfSquare = area(5, 5); // areaOfSquare contains the value 25

    var anotherArea = area(10, 20); // anotherArea contains the value 200

  • You can pass a function as an argument to another function by using the name of the function as the argument. This essentially means that a function can be stored in a variable.

    function area(width, height) { return width * height; }

    function perimeter(width, height) { return width * 2 + height * 2; }

    function calculate(operation) { // operation is a variable pointing to a function that's passed in by the caller of the function var width = 50; var height = 50; var result = operation(width, height); // invoke the passed in function return result; }

    var a = calculate(area); // pass in area function as argument. notice we are not invoking area function at this point.

    var p = calculate(perimeter); // pass in perimeter function as argument

  • You can return a function from within another function.

    var add100To;

    function addBy(amount) { // We can nest a function within another function function add(num) { return num + amount; } return add; }

    add100To = addBy(100); // add100To points to the "add" function

    var sum = add100To(6); // sum is 106

    var anotherSum = add100To(25); // anotherSum is 125

  • Anonymous functions

    • Anonymous functions are functions with no names (or identifiers).

    • You can store an anonymous function using a variable and invoke the function by appending a pair of parentheses after the variable:

      var area = function (width, height) { // area is now a function return width * height; };

      var size = area(4, 5); // size will be 20

    • You can invoke an anonymous function by wrapping it with a pair of parentheses, and then adding another pair of parentheses after it. This is called an Immediately Invoked Function Expression (IIFE):

      // area is now a number var area = (function () { var width = 4; var height = 5; return width * height; })();

    • You can define parameters for an IIFE, just make sure you pass the proper arguments to the IIFE when invoking it:

      // area contains the number 40 var area = (function (width, height) { return width * height; })(4, 10); // passing two arguments to this IIFE.
